# Flight-Ticket-Booking
✈️ Flight Ticket Booking System

📌 Overview
A front‑end web application that simulates flight ticket booking using HTML, CSS, and JavaScript. Users can search flights, view schedules, select seats, and confirm bookings — all within a responsive, interactive interface.

🚀 Features
Flight Search: Filter flights by source, destination, and date.

Dynamic UI: Responsive design with CSS for mobile and desktop.

Seat Selection: Interactive seat map built with JavaScript.

Booking Confirmation: Instant ticket summary generated on screen.

Local Storage: Save booking details temporarily in the browser.

🛠️ Tech Stack
HTML5 → Structure and semantic layout

CSS3 → Styling, responsive design, animations

JavaScript (ES6) → Logic, DOM manipulation, data handling
